Bounce It:

Javascript Motion Basics

by Mrs. Kitson

How does one animate a bouncing ball so that the outcome is smooth and seems natural, yet at the same time without animating every frame by hand or creating some sort of physics element?

This can be accomplished through HTML and CSS, if interaction such as scrolling on a website, or touching a object with a mouse is desired than the addition of JAVASCRIPT is necessary. These markup and programming languages make up a majority of the web they are the building blocks.
